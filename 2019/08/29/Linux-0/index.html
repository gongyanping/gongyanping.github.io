基于阿里云或虚拟机的Linux操作系统搭建

end linux centos xshell 2019/08/29

基于阿里云

1. 注册阿里云账号

2. 购买云服务器ECS

3. 选择自己的计费方式，地域，实例，操作系统类别和存储盘等，进行购买。

4. 在控制台中可以看到自己的云服务器，进行登录和使用。

基于虚拟机

1. 安装VMware

因为这个虚拟机的最新版本是要收费的，我下载的是当前较新的版本14.13。
下载地址：https://pan.baidu.com/s/1IrrjGq4PdKap6cbLbTFEuA
提取码：a59o
VMware的安装很简单，只需下一步下一步就可以了。

2. 安装操作系统

可以在官网上免费下载，
下载地址：https://www.centos.org/download/ 选择DVD ISO.
DVD ISO 和 Minimal ISO 的差别是后者是简化版。
在VMware上新建虚拟机并安装Centos的操作过程随意度娘即可。

3. xshell和xftp下载安装

下载地址：https://pan.baidu.com/s/1YsSI7KIcHdhxSg89SDc0TA
提取码：uqdu
解压到相应盘，各自运行exe即可。
